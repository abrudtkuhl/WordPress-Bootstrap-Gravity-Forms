WordPress-Bootstrap-Gravity-Forms
=================================

A WordPress plugin to apply Bootstrap 3 styles to Gravity Forms plugin

Usage
-----
1) In Gravity Forms Settings, turn off "Output CSS"  
2) In Gravity Forms Settings, turn on "Output HTML5"  
3) Enable Plugin  
4) Drink A Beer
